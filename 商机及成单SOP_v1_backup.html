!核心问题与解决思路

🔴 当前问题

订单转成单周期平均 5 天，远高于同行 2-3 天。一线顾问逼单技巧因人而异，缺乏统一标准化流程。部分顾问（尤其新人）缺乏判断客户意向和主动逼单的意识。

🟢 解决思路

将销冠和资深顾问的"实战技法"技巧标准化，形成可抄作业的话术和动作指南。从商机转订单阶段就开始铺垫心理预期。不违反公司规则，话术尽可能"野"，以结果为导向。

关键洞察：真正的转化效率取决于在引导客户下单时铺垫了什么样的心理预期——用"抢房"而非"申请"，用"锁优惠"而非"下单"。

1阶段一：商机 → 高意向

🔰 高意向判断标准（新人必读）：只要学生没有明确说"不行""太贵了不考虑"，默认就是高意向。尤其对于 Accolade / Scape 等学生公寓——他只要没排斥，就是你的单。下了单就默认他要定，不付钱就是有问题，针对性解决即可。

🔑 一、开始沟通 —— 详细步骤（分渠道）

不同渠道进来的客户，开场方式不同。核心原则：先让对方有回复，再逐步挖需求。不追求一步到位。

沟通渠道优先级：能发语音 → 发语音；能打电话 → 打电话。文字聊天的信息密度和情感传递远不如语音。语音能更快建立信任、更快拿到真实需求。
📋 渠道客户（微信群/BD分配）—— 标准四步
1
进群第一分钟：发送开头语
"同学你好，我是一乡好居的XX，主要负责悉尼/墨尔本区域的房源对接。咱们这边有什么需求随时沟通~"
目的：让对方知道你是谁、你能做什么。话术要点：简洁、不啰嗦。
2
确认基本信息（低成本问题）
依次确认：① 学校/校区 ② 计划入住时间 ③ 性别 ④ 签证状态
注意：一次只问1-2个，不要连珠炮。学生回复一个再问下一个。
示例："同学你是哪个学校的呀？"→ 等他回 → "计划什么时候入住呢？"
3
制造"咨询火爆"的第一印象
在对话中自然植入："刚才有好几个同学也在问这个区域的房子"、"最近7月入住的同学特别多"
目的：让学生感受到房源紧俏，为后续逼单埋伏笔。同时暗示"你不是我唯一的客户，我有的是客户"——提升你在对话中的心理地位。
4
开始第一轮推荐
拿到基本需求后不要等，立刻推荐第一套房源。用"第一眼原则"选房（详见推荐章节）。
如果学生不回复 → 立即调整：换更简单的问题，或换房源特征重新吸引。
📱 小红书 / APP 客户 —— 差异化开场
1
假装搞混帖子来源
"同学你好，你是刚才问XX studio的那位同学吗？"（故意说错帖子内容）
→ 学生纠正："不是，我看的是2B2B"
→ 你："哦不好意思搞混了，今天问这个公寓的同学特别多"
效果：① 获取真实需求 ② 暗示房源火爆 ③ 自然打开对话
2
反向回小红书确认
"刚才在小红书加我问studio那个是你吗？" → 学生说不是 → "那可能搞错了，不过既然加了，你有啥需求也可以问我"
效果：降低学生的防备心理，同时暗示你客户很多。
3
用语音建立信任
小红书客户和APP客户通常戒备心较强、比价意识重。文字聊几句后，尽早切语音。语音能传递专业感和可信度，比文字更容易建立信任关系。
👨‍👩‍👧 家长客户 —— 信任优先
1
先展示专业度，不要急着推房源
家长和学生的决策逻辑不同——学生看颜值和距离，家长看安全和合规。开场先介绍公司背景、合作公寓品牌、服务流程，建立"正规机构"的认知。
2
主动要求拉群或电话沟通
"阿姨/叔叔，方便的话我直接给您打个电话，或者拉个群把孩子也拉进来，我一起给你们介绍，有什么问题随时问我。"
家长更信任语音沟通，且拉群后孩子在群里也能看到信息，减少信息传递损耗。
3
安全感和保障是核心卖点
对家长推房源时，优先强调：① 公寓品牌和资质 ② 24小时安保/门禁 ③ 退订政策保障 ④ 中国学生比例和评价。价格反而不是家长最关心的——他们更担心孩子住得不安全。
🦊 破冰实战技法补充
故意说错信息诱导纠正："你是悉尼大学的吧？"→ "不是，我是新南的"→ 获取真实信息，同时自然打开对话 明知故问："咱是7月份入住是吧？"（群名已标注7月→再次确认，让他回复） 非渠道客户（小红书/APP）：假装搞混——"你是刚才问XX房源的同学吗？"暗示该房源咨询火爆，同时探出他真实感兴趣的内容

🔎 二、探需求 —— 详细方法论

学生往往不知道自己要什么，但一定知道不要什么。探需求不是一次性问完，而是推-挖-推循环：推一套 → 看反馈 → 挖出偏好 → 再推更精准的。

🔄 推-挖-推 循环法
推
发一套特征突出的房源
只发最核心卖点（如"走路5分钟到学校"），不要发全套资料。用留白式推荐激发学生提问。
看
观察学生反馈
他问什么 = 他看重什么。问价格 → 预算敏感；问距离 → 通勤优先；问房间大小 → 居住体验优先；只回"好的" → 不感兴趣，换方向。
挖
顺着他的问题深挖
学生问"离学校多远？"→ 你反问"你希望走路多久以内？10分钟还是20分钟？"
学生问"多少钱？"→ 你反问"你的预算大概在什么范围？"
用封闭式问题引导学生说出具体需求。
推
根据反馈精准再推
循环回到第一步。每轮推荐都比上一轮更接近学生的真实需求。一般 2-3 轮循环即可锁定需求。
🔬 关键信号识别（学生说了什么 = 他在想什么）
学生说什么真实含义你的应对
"你们是先交定金还是先签合同？"正在其他平台对比，别人让他交定金了立刻追问："你在别的地方也在看吗？那边什么条件？"了解竞品情况，针对性对比
"这个我再看看"有顾虑但没说出口直接问："是价格不合适还是位置不合适？"用封闭式问题引导他说出真实顾虑
"我爸妈要看看"决策权不完全在他手里，或他在用家长当借口拖延"方便的话拉个群，我直接跟叔叔阿姨介绍，他们有什么问题我直接解答"
"这个还不错"高意向信号！有兴趣但不急立刻跟进："要不要先下单锁一下？随时可以取消的"
"太贵了"可能是真的超预算，也可能是在试探底线"你的预算大概多少？我帮你找找有没有合适的"——先确认是真超预算还是假超预算
只回表情包或"好的"不感兴趣或没时间看不要追问，换个特征完全不同的房源再推一次
核心原则：对学生产生好奇心。他说每句话你都要想"他为什么这么说？"。不要学生说什么你执行什么——学生说的需求 信一半就行，需求是可以被引导和改变的。最重要的是通过他反馈的线索，锁定他真正在意的点。

🏠 三、精准推荐与挖掘需求

  • 1推荐顺序（必须遵守）：独家房源、必住榜房源 → 佣金高、好卖的房源。该顺序直接关联奖金收益。
  • 2"第一眼原则"：第一套房源宁可超出预算，也要选视觉冲击力强、卖点突出的房子，抓住学生眼球。实战技法
  • 3留白式推荐：不要一次性把所有信息（地图、视频、价格）都发完。只发最关键的一个卖点，吸引学生主动提问。他问什么 = 他看重什么。实战技法
  • 4不做无效推荐：学生对这套有兴趣，就决不推第二套。推多了他反而选择困难。
📋 推荐房源三要素（必发）
① 实拍图片 / 视频 ② 地图周边配套 ③ 人话卖点 ≤ 20字
不要一次性把所有信息都发完。只发最强卖点吸引学生提问。拒绝 AI 长文案和 emoji 堆砌，用人话短句。

🔄 四、商机回访策略

商机回访是激活存量、提升效率的必备动作。前期已有沟通基础的客户，回访成交成本更低、概率更高。三类回访方式可组合使用：

  • 1聊天框置顶：把待回访客户置顶，从下往上 = 回访时间倒序。最久没联系的排在最上面，优先跟进。
  • 2专属标签分类：给客户打标签（时间维度：7月入住/9月入住；需求维度：Studio/2B；学校维度：USYD/UNSW），方便批量筛选精准回访。
  • 3导表批量跟进：日常统一安排导表，个性化需求主动向上级申请。注意：导表后不要直接在微信搜名字——群昵称和真实姓名不一致会导致搜不到。正确做法：一个窗口开商机表，一个窗口开CRM商机列表，对照查找精准锁定。
  • 4回访时务必 @客户：日常消息繁杂，不@大概率被忽略。尤其4-6月、7-9月非紧急需求期，不@几乎无法触达。

🔍 五、查房技巧

  • 1群内搜地址：直接搜房源地址查看历史信息（地址必须完全一致）
  • 2拼房群提问：你不知道的信息大概率有人知道。同样适用申请流程、政策等

📮 六、公寓反馈技巧

  • 向公寓反馈问题时，带上学生名字 + 邮箱，公寓才能快速定位

📵 七、沟通不回复 / 敷衍的应对方法

大概率推荐未满足需求。从某一特征突出的房源入手，逐步锁定痛点：

次序策略参考话术
第1次正常推荐—
第2次发价格极低房源"目前新南有一套价格特别优惠的房源，发给你参考下~"
第3次发位置极近房源—
第4次发装修精致、颜值高房源—
第5次发租期灵活房源—
均无效大概率无实际找房需求 → 降低回访频率，定期跟进

2阶段二：高意向 → 订单

客户小程序下单 → 成单概率已达 70%

🧭 一、下单思路决策树

看到学生的不同信号，用不同的逼单角度。不是所有学生都吃同一套。

学生信号用哪种逼单角度核心话术方向
对房源表示喜欢，但说"再看看"稀缺性 + 零风险"这间房只剩最后几间了，要不先下单锁一下？随时可以取消的，没有任何费用"
在几个房源之间犹豫给建议 + 锁优惠"我建议你先选XX，因为…（给理由）。今天的返现活动最后一天，先下单把优惠锁住"
担心价格/预算优惠 + 零风险 + 对比"这个价格在这一片已经是最低的了。你先下单占个位置，后边有更合适的随时换"
嫌远/嫌户型不好先确认是真嫌弃还是假借口反问："如果距离近一些/户型好一些，你会考虑吗？" → 如果说是 → 换房源。如果说"再说吧" → 他根本没意向，降低跟进优先级
自称"不着急"租期匹配 + 抢房"适合你开课时间的房源本来就少，现在不定后边可能没了"
问很具体的问题（合同/定金/流程）默认成交 + 封闭提问他在对比平台，是强意向信号。直接推下单："信息我发你了，咱们先把单子下了，后边的细节我帮你跟进"
下单思路总结：根据不同信号选不同角度——① 房源稀缺 ② 优惠即将截止 ③ 租期匹配稀缺 ④ 零风险无损失。关键是不要等他做决定，你替他做决定——"我建议你先锁这个，因为…"
⏰ 什么时候该放弃这条商机？不是所有商机都值得无限跟进。满足以下 3 个条件中的 2 个，降低优先级，转为定期回访而非主动逼单：
① 连续 3 次推荐不同特征房源，学生均无实质反馈（只回"好的""看看"算无反馈）
② 学生明确说"暂时不找了""签证还没下""学校还没定"——需求时间窗口不在当前
③ 付款阶段用了 3 种以上逼单角度仍不付，且无合理解释
原则：精力花在最有成交概率的客户身上。放弃不是失败，是资源优化。

📱 二、Push 申请核心策略

  • 1制造"抢房"氛围："这间房只剩最后几间了，很多人在问。咱们是'抢'一下，还是'申请'一下？"——用"抢"不用"申请"。实战技法
  • 2公寓截图施压：利用与公寓方的沟通截图（或找人配合演的截图），告诉学生："公寓老师刚催了，说再不定就要清掉名额了。"实战技法
  • 3零风险占坑："现在下单只是占个位置、锁住优惠，随时可以无责取消，没有任何费用和信用影响。"必须说清楚零风险，这是扫除下单心理障碍的关键。
  • 4优惠/租期双重施压："今天的返现活动是最后期限，先下单锁住优惠" + "这间房的起租时间非常适合你的开学时间，非常稀缺"实战技法
📝 Push 申请标准话术（可直接复制）
"下单无任何损失，后期若不需要该房源，可随时取消" "当前下单可暂时锁定房源位置，避免后期心仪房源售罄" "确认无误后再付款即可"
关键认知：Push 下单不要求当场成交，而是：① 提升关注度 ② 提前收集信息（后续换房无需重收）③ "offer即将过期"是后续 push 付款的有力武器 ④ 下单房源作为后续对比辅助加快决策

3阶段三：订单 → 成单（付款 + 看房）

核心逻辑：默认已提交申请的学生就是要定的。立刻引导付款，直到他明确表示不付款。

💰 一、付款全流程时间轴（一条龙逼单）

0分
申请成功 → 立刻恭喜 + 发邮件
"恭喜！申请已经成功了！我已经给你发了邮件，你查收一下。"
关键：不要给学生冷静的时间，立刻进入下一步。
2分
制造紧迫感 → 迫使他立刻登录
"邮件里的更改密码链接2小时内有效，你尽快点进去改一下密码。"（实际可能48小时，但说2小时能迫使他现在行动）
关键：他既然申了，现在必然有时间。趁热打铁。
5分
确认信息 → 让他回复"没问题"
"登录进去了吗？确认一下租期和租金有没有问题，没问题跟我说一声。"
关键：让他亲口确认"没问题"，后续付款时他无法反悔说信息有误。
8分
发付款步骤图 → 降低操作成本
提前准备好每一步的截图：点哪里、填什么、怎么确认，全部标注清楚。甚至将支付宝/微信二维码直接下载好发过去。
关键：操作步骤越少越好。学生每多一步操作就可能产生变数。
10分
封闭式提问 → 默认成交
不问"你付不付？"，问"你是用澳币还是人民币付？"
不问"你能不能付？"，问"你现在方便用哪种方式？"
关键：默认他已经要付了，只差选方式。
为什么操作成本越低越好？只要学生不付款，任何变数都可能发生——家长说不好、朋友推荐别的、刷到差评……中间多一分钟就多一分风险。付款链接直接发、二维码直接给，让他扫一下就能付。
⚠️ 学生公寓 vs 社会公寓付款差异：以上一条龙流程主要针对学生公寓（改密码→登录→确认→付款）。社会公寓付款路径不同：可能是转账、扫码、飞汇（Western Union），没有统一的"改密码"步骤。社会公寓付款关键：① 确认好付款方式后直接发链接/二维码 ② 转账类付款做好汇款凭证跟进 ③ 社会公寓付款通常没有学生公寓那么紧迫的时间窗口，但同样遵循"越快越好"原则——学生拖得越久越容易变卦。

🦊 二、付款障碍 & 连环逼单招

📋 三、付款前必做

付款 = 成单 + 客诉起点。付款前必须确认以下两项，从源头规避纠纷：
  • 1退订政策再次确认：付款前务必再次发送退订政策，客户确认无误后再引导付款
  • 2手续费提前说明：跨境/平台付款手续费 1%~4%，明确告知费用由银行/支付机构收取，非我司费用
📝 付款提醒话术（可直接复制）
温馨提醒：因涉及跨境汇款，银行及支付平台会收取1%～4%不等的手续费（具体以银行费率为准），该费用由银行/支付机构收取，并非我司费用。（主要发生在信用卡、支付宝、微信等平台） 建议抵达悉尼后尽快办理当地银行卡，用本地卡前往前台换绑，可免除手续费，更安全也更划算。

🏃 四、邀约看房

看房时机判断：看房不是只能在付款前——两种时机都可以。① 付款前看房：学生有顾虑、不确定房源实际情况时，先看房再付款，看完心理从"不确定"→"确定"。② 先下单付款再看房：适用于房源紧俏、学生对公寓品牌已信任的场景——先锁房再看，不满意再退。学生公寓优先走②（先下单再看），社会公寓两种都可以。

💡 校园大使转化核心技巧
  • 1多次触达 > 完美话术：学生不推往往不是不愿意，而是忘了、不知道能赚钱、不知道怎么操作。多次提醒是转化的关键。
  • 2续租免服务费换转介绍：学生续租时："这次服务费我不收了，你帮我多推几个同学过来就行。"学生大概率会答应，且会把你的名片发新生群/家长群。
  • 3直接要求 > 委婉请求：对于关系好的学生，可以直接说："你得帮我推两个朋友过来。"他反而会觉得你们关系到位。
  • 4订房后立刻拉专属服务群：把学生、你自己、主管拉在一个群里——以后所有售后问题在群里解决，降低你的售后负担，同时让学生感到被重视。
  • 5利用学生的社交网络：一个学生把你的名片发到新生群/家长群，可能一夜之间加几十个人，且这些人的信任度极高（因为是熟人推荐）。
